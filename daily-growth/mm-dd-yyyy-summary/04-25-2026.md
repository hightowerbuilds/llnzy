### Completed: Per-Theme Shader Control (effects_on_ui flag)

  - `cancel_editing_tab()` — cancel without saving
  - `set_tab_context(tab_count, active_tab)` — update tab context before render
  - `take_saved_tab_name()` — retrieve pending name change
- Implemented double-click detection using time-based approach (300ms window)
- Added egui Area overlay with TextEdit when editing (positioned over the tab)
- Added Enter/Escape key handling
