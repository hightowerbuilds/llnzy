## Tab Context Menus

Improved the right-click tab menu workflow for joining tabs.

Completed:

- Kept the primary tab context menu anchored directly below the clicked tab.
- Changed `Join Tabs` to open its target menu as a side popup to the right of the primary menu instead of overlapping it.
- Kept outside-click dismissal aware of both popup rectangles so interacting with either menu does not close the menu unexpectedly.
- Removed the extra-click delay so users can right-click a tab, click `Join Tabs` once, and immediately see the side menu.

---

## Joined Terminal Scrolling

Fixed terminal scrollback routing when terminal tabs are part of a joined layout.

Completed:

- Added joined-terminal pane hit testing for the current pointer position.
- Routed mouse-wheel scrollback to the terminal tab under the cursor instead of always using the active tab.
- Routed mouse-reporting wheel events to the targeted joined terminal tab with pane-relative row/column coordinates.
- Preserved egui-owned scrolling for Markdown and code panes.

This addresses cases where:

- Terminal + Markdown layouts left the terminal unable to scroll when the Markdown pane was active.
- Terminal + CodeFile layouts had the same active-tab routing problem.
- Terminal + Terminal layouts could fail to scroll either shell reliably.
