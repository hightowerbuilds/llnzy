# 05-07-2026 Summary

## Shortlist Implementation Pass

Worked through the active shortlist items and shipped a set of app polish and Sketcher improvements.

Completed:

- Added app-level Cmd+Plus, Cmd+Minus, and Cmd+0 font zoom behavior so keyboard zoom changes LLNZY text sizing instead of the native window scale.
- Restored terminal appearance behavior in built app flows, including background image resolution from the background library and CRT/effect rendering without adaptive-quality silently disabling requested effects.
- Improved the sidebar tree with safer folder connector padding, hover highlights, live filesystem refreshes, outside-app file drops, and immediate rendering after folder expansion.
- Improved Markdown preview layout with a right-side gutter for narrow joined tabs and a grabbing cursor during drag-to-scroll.
- Removed Kill Terminal and Restart Terminal actions from the tab context menu.
- Fixed tab-name editing behavior:
  - Edit state now visibly blinks.
  - The existing title is selected from the front so new typing replaces it instead of concatenating.
  - The right-click menu click no longer immediately closes the rename editor.

## Sketcher Work

Extended Sketcher into a more useful planning surface.

Added:

- Image import into sketches, copied into a Sketcher image library so saved documents can keep resolving imported images.
- Automatic large-image fitting plus a selected-image resize slider.
- Canvas zoom controls from 50% to 200%.
- A developer symbol palette with database, table, API, server, queue, cache, cloud, lock, user, component, decision, and flow symbols.
- SVG export to the repo root or a user-selected folder.
- Sketcher keyboard commands:
  - Cmd+Z for undo.
  - Cmd+Shift+Z and Cmd+Y for redo.
  - Cmd+V to paste clipboard text into the active text edit or onto the canvas as a new text box.
