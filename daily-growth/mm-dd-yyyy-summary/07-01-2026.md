# Evening Session

## Breadcrumbs

**Dev instance tooling**
- Added `LLNZY_PROFILE` support to `PlatformPathSet`: every platform path
  (config, data, cache, themes, workspaces, logs, prompts, recovery) moves
  from the `llnzy` app dir to `llnzy-<profile>`, isolating a dev build from
  the daily driver. `./dev.sh` launches with `LLNZY_PROFILE=dev`.
- Fixed plain `cargo run` (broken with four binaries defined) via
  `default-run = "llnzy"`.

**Terminal: Claude Code scroll fix**
- Diagnosed by capturing Claude Code's startup escape sequences and grepping
  its binary: it enables SGR mouse reporting (`?1000h`/`?1006h`) and expects
  wheel events delivered as mouse reports so it can scroll its own transcript.
- New `terminal/mouse.rs` model: wheel routing (mouse report / alternate
  scroll arrows / local scrollback; Shift always scrollback) plus SGR and
  legacy X10 encoders. Surface accumulates fractional trackpad deltas
  instead of rounding them away.

**Explorer live refresh**
- New `fs_watch` module: recursive `notify` watcher on the workspace root,
  200ms debounce, `.git` and collapsed-directory churn filtered by a pure,
  tested relevance function.
- Workspace polls the watcher every 250ms, invalidates the explorer tree
  cache, and re-renders; sidebar file ops (rename, create, delete,
  drag-move, image import) invalidate instantly.

**Explorer context menu placement**
- Menus estimate per-view height and anchor their bottom edge at the click
  point (open upward) when they would overflow the window bottom;
  horizontal position clamps at the right edge.

**Editor: word wrap rebuilt**
- Wrap points now break after the last whitespace in each window instead of
  fixed-column chunks; one `line_wrap_starts` function feeds visual rows,
  row counts, cursor motion, and hit-testing so they cannot disagree.
- Wrap width uses a measured average glyph advance for the active font
  (shaped once per font/size) instead of the 0.6-em guess — the root cause
  of markdown's clipped/early wraps under the prose font.
- Vertical cursor movement clamps inside short wrapped segments.

**Markdown preview styles**
- New `MarkdownPreviewStyle` (Default / Newspaper / Research Paper),
  persisted as a preference, selectable in Appearances > Editor, rendered
  through a per-style typography table in `render/markdown.rs`.
- Newspaper: researched from NYT production CSS, Guardian source tokens,
  and WaPo design tokens — Georgia body, two-tone ink hierarchy, hairline
  rules, 600px measure, pull-quote blockquotes.
- Research Paper: verified LaTeX article-class spec — Charter body, the
  1.2x size ladder, centered non-bold title, auto-numbered sections,
  abstract-style quotes.
- All style colors derive from the active app theme (light/dark follow);
  this also removed the last hardcoded preview code-block colors.

**Desktop menu cleanup**
- Editor menu trimmed 15 -> 5: explicit markdown modes collapsed into
  Cycle; all nine LSP entries removed in favor of conventional keybindings
  (F12, Shift+F12, F2, Ctrl+Space, Cmd+., Alt+Shift+F, Cmd+Shift+O) and the
  existing command-palette coverage.
